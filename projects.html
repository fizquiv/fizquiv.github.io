Offgrid

Offgrid is a React Native app that enables hikers to stay connected in remote areas without cell service. The app pairs with an ESP32 device over Bluetooth Low Energy (BLE), which relays text messages via long-range LoRa signals. This creates a peer-to-peer texting experience that works entirely offline, allowing hikers to coordinate safely in environments where traditional communication tools fail. I developed the mobile app, which provides a familiar chat interface and manages BLE connections, while a teammate handled the ESP32 firmware. Offgrid currently supports text messaging, and its design emphasizes reliability and simplicity for outdoor use. Planned features include location sharing, group chats, and SOS alerts‚Äîcapabilities that go beyond what traditional walkie-talkies offer, such as real-time location tracking for safety.

Shattered Timeline

Shattered Timeline is a roguelite top-down game built with JavaScript on the HTML5 Canvas API, featuring a fully custom game engine developed from scratch by a three-person team. The engine handles rendering, collision detection, player movement, combat logic, and UI ‚Äî all without relying on external game frameworks. The game includes a player account system with login and authentication, backed by a Node.js + Express server connected to a MySQL database. Player stats, progression, and unlocked weapons are saved persistently, and the repository provides setup instructions for hosting the database locally. Gameplay features include five levels, a store system, and boss fights on each of the first three floors. Development was managed with Git/GitHub, with each teammate working in separate branches. I contributed primarily to the engine architecture, combat system, and user interface, while also supporting the database design. Future improvements include the completion of enemy sprite animations to enhance the visual experience.

Technologies Used:
JavaScript(ES6) HTML5 Canvas API Node.js + Express MySQL Git/GitHub
