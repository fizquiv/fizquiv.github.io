Offgrid

A React Native app enabling offline communication in remote areas without cell service. The app pairs with an ESP32 device via Bluetooth Low Energy (BLE) to relay text messages through long-range LoRa signals, creating a peer-to-peer texting experience. I developed the mobile app with a chat interface and BLE connection management, while a teammate built the ESP32 firmware. Planned features include location sharing, group chats, and SOS alerts for enhanced outdoor safety.

Technologies Used:
ESP32 LoRa React Native BLE

3D Multi-Agent Traffic Simulation

A real-time 3D visualization of a multi-agent traffic system built with WebGL and vanilla JavaScript. Features autonomous agents (butterflies) navigating a city with dynamic traffic lights and obstacles. Uses Python Flask backend for pathfinding and collision avoidance via REST API. The 3D graphics pipeline includes custom GLSL shaders with Phong lighting, up to 100 dynamic point lights, smooth interpolation, procedurally animated wings, and texture-mapped OBJ models. Includes real-time statistics and simulation controls.

Technologies Used:
WebGL JavaScript (ES6) GLSL Shaders Python Flask REST API 3D Graphics Multi-Agent Systems
